Projeto Dio

Neste projeto, o objetivo é criar uma página com links, explicando o que foi aprendido durante o curso.

Índice

  • História da web
  • O que são clients
  • O que são Servers
  • Linguagens de programação
  • Introdução ao HTML na prática

História da web

(Voltar)

O módulo trouxe informações sobre o surgimento dos primeiros computadores e grande nomes da tecnologia.

O que são Clients

(Voltar)

O módulo trata sobre o que são "clients" que é o lado cliente (dispositivo como computador, celular, etc). Enfatizando que devemos ter boas práticas e realizar testes. Também exemplifica quais são os diversos navegadores, e para finalizar, fala sobre os dispositivos móveis (deve se atentar ao Design responsivo e responsividade), pois o número de dispositivos móveis só tende a aumentar, então, é necessário levar isso em consideração quando for criar uma página.

O que são Servers

(Voltar)

Neste módulo, aprendemos que os Servers "servem" informações para o cliente através do banco de dados (armazena). Também vimos as diferenças de softwares e hardwares dos servidores e os métodos de hospedagem dos sites e seus domínios

Linguagens de programação

(Voltar)

O módulo trouxe a definição de linguagem de programação, exemplos de liguagens server-side e client-side, lembrando que HTML não é linguagem de programação, e sim um arquivo de marcação.

Introdução ao HTML na prática

(Voltar)

Por último, o módulo prático nos traz informações sobre as ferramentas que iremos utilizar para começar a usar o HTML, mostra o inspetor de elementos (que é fornecido pelo navegador) e aborda a estrutura básica do HTML, e seus atributos de Tags, ferramentas de texto, listas e links.
